## Sugestão de fala

Olá, meu nome é Caio Augusto C. G. e este é o TP3 do projeto Dashboard de Sustentabilidade, relacionado ao ODS 13, ação contra a mudança global do clima.

Nesta sprint, foi desenvolvido o primeiro entregável funcional do sistema.

Primeiro, no backend, foi criada uma API em Node.js com Express. A API retorna dados simulados de emissão de CO₂ por país e ano. Também foi criada uma rota para listar os países disponíveis.

Agora, no frontend, foi criada uma aplicação em React. A interface apresenta um dashboard com filtro por país, gráfico de emissão de CO₂, tabela com os dados e botão para exportar os dados em CSV.

Ao selecionar um país, o frontend faz uma requisição ao backend e atualiza os dados exibidos. Também é possível exportar os dados apresentados em um arquivo CSV.

No GitHub, o repositório está organizado com as pastas backend, frontend, docs e Videos. A documentação do TP3 e o planejamento do TP4 foram adicionados na pasta docs.

Para o TP4, estão planejadas melhorias como implementação de login, integração com API externa real de dados climáticos, filtro por ano e melhorias na interface.

Esse é o estado atual da solução desenvolvida no TP3.
