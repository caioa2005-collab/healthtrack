# Documentação Final do Projeto

**Projeto:** Dashboard de Sustentabilidade  
**Aluno:** Caio Augusto C. G.  
**ODS:** ODS 13 – Ação contra a mudança global do clima

## Objetivo

Desenvolver um dashboard web para visualização de dados de emissão de CO₂ por país e ano.

## Problema

Informações sobre emissão de CO₂ podem ser difíceis de interpretar quando disponibilizadas em relatórios extensos ou planilhas. O sistema busca facilitar essa visualização.

## Proposta de solução

Aplicação web com frontend em React e backend em Node.js. O sistema permite login, consulta de dados, filtros, gráfico, tabela e exportação CSV.
