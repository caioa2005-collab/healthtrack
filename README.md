# Projeto ODS: Dashboard de Sustentabilidade

## ODS Escolhido
ODS 13 – Ação contra a mudança global do clima

## Objetivo do Projeto
Desenvolver um dashboard web interativo que permita visualizar dados de emissão de CO₂ por país, promovendo conscientização ambiental.

## Tipo de Solução
Dashboard web com backend em Node.js e frontend em React, garantindo visualização dinâmica e armazenamento seguro dos dados.

## Funcionalidades Principais
- Login de usuário
- Visualização de gráficos de emissão de CO₂
- Exportação de dados em CSV
